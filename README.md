# AI Startup Factory

Scaffold for an AI startup generation platform with a FastAPI and LangGraph backend, React and Vite frontend, and Docker Compose deployment.

## Included

- FastAPI API with websocket status endpoint
- LangGraph workflow skeleton with agent nodes
- Redis memory, ChromaDB RAG, MLflow forecasting hooks
- React dashboard with agent panels and a flow view
- Docker Compose services for api, frontend, postgres, redis, chromadb, and mlflow
